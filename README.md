# Homestead for Fish Shell

## Install

### Using [Fisher](https://github.com/jorgebucaran/fisher)

```fish
fisher add  dwhoban/fish-homestead
```


## Usage

Pass any `vagrant` command to your global Laravel Homestead machine.  Comes in two flavours:

```fish
homestead
```

or

```fish
hs
```
